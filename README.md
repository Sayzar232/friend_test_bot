# Тест на Дружбу - Telegram Бот

Этот Telegram-бот на основе `aiogram 3` позволяет пользователям создавать викторины о себе и делиться ими с друзьями, чтобы проверить, насколько хорошо они их знают.

## 🚀 Функционал

- **Создание викторины**: Пользователи могут создать свой собственный тест с вопросами о себе.
- **Прохождение теста**: Друзья могут проходить тест по уникальной ссылке.
- **Подсчет результатов**: Бот автоматически подсчитывает правильные ответы и показывает результат.
- **Админ-панель**: Базовые команды для администратора бота.

## 🛠️ Технологии

- Python 3.10+
- aiogram 3
- PostgreSQL (или другая асинхронная БД, поддерживаемая `asyncpg` или аналогичными драйверами)

## ⚙️ Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL вашего репозитория>
    cd friend_test_host
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Windows: venv\Scripts\activate
    ```

3.  **Установите зависимости** (предполагается наличие файла `requirements.txt`):
    ```bash
    pip install -r requirements.txt
    ```

4.  **Настройте переменные окружения:**
    Создайте файл `.env` в корневой директории проекта по примеру `settings.py` и добавьте в него следующие переменные:
    ```env
    BOT_TOKEN="ВАШ_ТЕЛЕГРАМ_БОТ_ТОКЕН"
    ADMIN_ID="ВАШ_ТЕЛЕГРАМ_ID"
    DATABASE_URL="postgresql+asyncpg://user:password@host:port/dbname"
    ```

5.  **Запустите бота:**
    ```bash
    python main.py
    ```
